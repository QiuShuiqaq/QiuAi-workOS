# Web Console

Main browser console for Personal Workspace and Enterprise Workspace.

Frontend direction:

- Next.js
- React
- Ant Design
- Ant Design Pro layout patterns
- Ant Design X for AI interaction surfaces
- `@qiuai/ui` for QiuAI business components
- `@qiuai/api-contract` and `@qiuai/api-client` for backend communication
