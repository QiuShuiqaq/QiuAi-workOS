# Web Console

Main browser console for personal users and enterprise control users.

Responsibilities:

- login and workspace switching
- personal account and billing views
- enterprise overview
- organization and department administration
- role, task, approval, and cost visibility
- desktop binding and runtime summary visibility

Frontend direction:

- Next.js
- React
- Ant Design
- Ant Design Pro layout patterns
- Ant Design X for AI interaction surfaces
- `@qiuai/ui` for QiuAI business components
- `@qiuai/api-contract` and `@qiuai/api-client` for backend communication
