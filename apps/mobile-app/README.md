# Mobile App

Mobile app direction: React Native + Expo.

Rationale:

- Strong React and TypeScript alignment
- Mature mobile development workflow
- Easier shared domain/API code reuse with Web and Electron
- Better fit for future native push, notifications, camera, file upload, and approval flows

Mobile UI should not copy Ant Design desktop layouts. It should reuse domain contracts and API SDK, while using mobile-appropriate components.
